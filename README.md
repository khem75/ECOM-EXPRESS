# 📦 Ecom Express Sales Dashboard

An interactive Power BI dashboard built to analyze E-commerce sales performance across products, categories, and states. The dashboard provides insights into revenue trends, sales metrics, customer behavior, and product performance through dynamic visualizations.

## 📊 Key Features
- Revenue, Sales, AOV & Cancellation Rate KPIs
- Revenue by Product
- Revenue by Category
- Revenue by State
- Daily Revenue Trend Analysis
- Interactive Category Navigation
- Business Performance Insights

## 🛠️ Tools Used
- Power BI
- DAX
- Power Query
- Excel/CSV Dataset

## 📸 Dashboard Preview
- Sales Performance Dashboard
- Product & Category Analysis
- Revenue Trend Dashboard
